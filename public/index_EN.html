3D Video Game "EIF MUZIEJUS"

About Project

The aim of the project is to create a 3D video game of Vilnius College Faculty of Electronics and Informatics (EIF):

Create an interactive museum with the ability to listen audio recordings of museum exhibits and display a detailed description (In progress...). Design 3D models: classrooms, interior and exterior and other buildings. Already done: reading room, library, software department, classroom 202, "Simonas Daukantas" gymnasium (building facade), Seimas hotel (building facade), etc. Share assets for students: 3D models, audio recordings, description/titles text and other materials (Any suggestions are welcome if you have any idea). Final goal create "EIF-MUZIEJUS" exibitons web service that allows students to create applications (web, mobile, etc.).

Free demo version you able to try now!
